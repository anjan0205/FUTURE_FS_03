# NE FITNESS UNISEX GYM — Premium Fitness Website

A stunning, mission-critical gym website built with a **dark industrial-military aesthetic**. Designed to wow users with high-energy visuals, custom branding, and sophisticated animations.

![NE FITNESS Website Preview](images/hero-bg.png)

## 🚀 Live Demo
The project is a static high-performance single-page application.
- **GitHub Repository**: [https://github.com/anjan0205/ne-fitness-gym](https://github.com/anjan0205/ne-fitness-gym)

## 💎 Key Features
- **Premium Aesthetics**: Dark mode design with red-orange accents, vibrant glows, and industrial typography.
- **Cinematic HQ Imagery**: Custom generated 4K-quality photography for all service categories.
- **Interactive Animations**:
    - **Canvas Sparks**: Physics-based background particle system.
    - **Glitch Reveal**: Multi-layer chromatic aberration on main headlines.
    - **Staggered Entrance**: Letter-by-letter 3D text reveals on load.
    - **Scroll-Aware Layout**: Fade-in sections and animating stat counters using `IntersectionObserver`.
- **Fully Responsive**: Mobile-first architecture with a dedicated navigation overlay.
- **Brand Integrated**: Custom logo integrated into the header, footer, and favicon.

## 🛠️ Built With
- **HTML5**: Semantic structure.
- **CSS3**: Custom design tokens, variables, and complex @keyframe animations.
- **JavaScript**: Performance-optimized vanilla JS for physics and scroll effects.
- **Fonts**: [Bebas Neue](https://fonts.google.com/specimen/Bebas+Neue) & [Barlow Condensed](https://fonts.google.com/specimen/Barlow+Condensed).

## 📥 How to Clone and Run Locally

1. **Clone the repository**:
   ```bash
   git clone https://github.com/anjan0205/ne-fitness-gym.git
   ```

2. **Navigate to the directory**:
   ```bash
   cd ne-fitness-gym
   ```

3. **Run a local server**:
   Since it's a static site, you can use any live server. If you have Node.js installed, you can use `serve`:
   ```bash
   npx serve .
   ```

4. **Open in Browser**:
   Open `http://localhost:3000` (or the port specified by your server).
